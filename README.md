<!-- HOW IT WORKS -->
<div style="background-color: #1e2338; border-radius: 16px; padding: 15px; margin: 20px 0;">
    <h2 style="color: #FFD93D; margin: 0 0 10px 0;">🧠 How It Works</h2>
    <p><strong>Collaborative Filtering using Cosine Similarity</strong></p>
    <ol style="line-height: 1.6; padding-left: 20px;">
        <li>User–book rating matrix created (706 books × 810 power users)</li>
        <li>Empty ratings filled with 0</li>
        <li>Cosine similarity computed between all book vectors</li>
        <li>For a selected book, system returns top N similar books</li>
        <li>Similarity score (0–100%) shows match percentage</li>
    </ol>
    <p><strong>Popularity Filter</strong> – Books sorted by rating count (≥250 ratings), then by average rating.</p>
</div>
